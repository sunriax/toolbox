# Toolbox for elmProject

*The Toolbox is a software to program and configure the elmProject!*
*It is written in C# (Windows Interface) and C (Projectloader).*
*This software should support an easy configuration of the system.*
*There are no additional Tools needed*

## Hardware:
	1. elmProject
	1. Arietta G25 (Linux Mikrocomputer)
	1. STK500/Megacard (ATmega16 Evalution Board)
	1. RS232/TTL Converter (FTDI FT232)
	1. XMOD FTDI JTAG Adapter (TE0792)
	1. Logic Converter 5V/3V3
	1. Wireless Access Point

## Software:
	1. Toolbox
	1. Loader
